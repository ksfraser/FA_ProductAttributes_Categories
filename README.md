# FA_ProductAttributes_Categories

Product Categories plugin for the FA_ProductAttributes module.

This plugin extends the core FA_ProductAttributes module with category management functionality, allowing products to be organized into hierarchical categories.

## Features

- Category creation and management
- Hierarchical category structures
- Product-to-category assignments
- Bulk category operations
- Category-based filtering and organization

## Installation

This plugin requires the FA_ProductAttributes core module to be installed first.

1. Ensure FA_ProductAttributes is installed
2. Copy this plugin to your FrontAccounting modules directory
3. The plugin will be automatically detected and loaded by the core module

## Development

This is a development directory for the categories plugin. The actual plugin code should be committed to the `plugins/fa_product_attributes_categories` submodule.
